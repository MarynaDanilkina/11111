## Getting Started

- Install node v22.13.1
- Install dependencies:

```bash
npm i
```

- Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## For build

- Run the command:

```bash
npm run build
```

- Run the assembled build:

```bash
npm start
```
